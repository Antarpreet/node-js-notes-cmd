# node-js-notes-cmd
A simple command-line app for managing notes.

Available command-line arguments:
```
Commands:
  add     Add a new note
  list    List all nodes
  read    Read a note
  remove  Remove a note

Options:
  --help  Show help                                                    [boolean]
```

To run the app:
1. npm install in root directory.
2. node app.js [commands] [-options]
